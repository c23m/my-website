# 地图存储格式指南

下面是地图存储格式：01.level
名称：纯数字
后缀：.level

## e
```json
{
    "name": "begin of levels",//名字
    "id": 1,//关卡标号
    "map": [//实体初始位置列表
        {
            "position": [//x,y,z坐标，不可缺省的值
                4,
                4,
                2
            ],
            "type": "player",//实体类型，默认"empty"
            "direction": 0,//初始方向,默认值0
            "inventory": [//物品栏，用列表,不填就是没有
                {
                    "type": "gem",
                    "count": 1,
                    "maxcount": 9
                }
            ]
        },
        {//第二个实体，显示了必要信息
            "position": [
                4,
                4,
                1
            ],
            "type": "empty",
            "direction": 0
        }
    ]
}
```

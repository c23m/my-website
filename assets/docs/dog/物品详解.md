

# 物品介绍
物品有类别和堆叠数量。实体持有某些物品会改变实体的状态等。  
物品对实体的影响写在实体的代码里,物品本身没有代码.  
实体不能有多个除了数量，其余完全一样的物品。
## 属性
> 改动过
```javascript
{
  type:"sword"//物品类型
  count:1//当前数量
  max:9//最大数量
  direction: [false, false, false, false] //方向
  tag: 0//标签
}
```
## 方向
个别物体会用得到。  
当某实体/物品需要具有多个方向时，使用这个属性。
>请在明显含方向属性的物品上标明“含方向”
## 标签
个别物体会用得到。  
当某实体/物品需要查找/绑定特定的实体（如踩下按钮使得远处的机关变动），使用这个属性。
>请在明显含标签属性的物品上标明“含标签”  
>“同标签的实体”指的是"含同标签[标识码]()的实体"

## 术语
损伤：实体损伤就会消失，但有些效果能避免  
移除：实体被移除就会消失，不可避免  
丢弃：某物品数量减少X个，0个就会消失。有些效果能避免
销毁：整组的物品，不论堆叠多少个，全部消失，不可避免

# 物品列表

## 信息类
包含信息的物品。
- 磁石
  - 有方向属性
  - 渲染在实体边界处
  - 默认为无方向(全False)，不渲染
- 标识码
  - 有标签属性
  - 1~15渲染成A~O，在方块左下角
  - 默认值是0，不渲染。
  > 实体有标签指的是有标识码，请修改相关内容！
- 宝石
  - 有标签属性
## 移动类
影响实体的移动。
- 弹簧
  - 若试图进入不可通行方块，改为反向[位移]()
- 磁干扰系统
  - 有方向属性
  - 实体不能向此方向移动
## 损伤类
可造成或避免损伤。
- 剑
  - 尝试推动实体时损伤该实体，若实体消失则进入该格
  - 不可堆叠（最大数量为1）
- 炸弹
  - 若实体的行动为[停留]()(包含位移回原位)，损伤自身
- 护甲
  - 在实体被损伤时，丢弃此物品并无效此损伤
  - 在实体右下角渲染成数字(0~9)，表示可减免损伤的次数，默认为1
- 电锯
  - 见实体[森林]()
- 气球
  - 在实体试图传递力信号时，将强度改为6

## 某类
- 钥匙
  - 见实体[门]()
- 安眠药
  - 回合开始时，[丢弃]()一个自身，并使拥有者本回合停留
- 高功率电池
  - 作用对象改为射线上的所有实体
- 遥控器
  - 实体若只能被力信号激活，且有标签，则接受强度为1的信号时会等同于接受到其他同标签的第一个实体的强度为7的力信号。
- 信号增幅器
  - >不一定加入
  - 实体行动后额外传递力信号给脚下实体周围临近的4个位置的实体
- 警报
  - 使此实体面对的实体无法被损伤
  - 或：使此物体的下一个物体无法被丢弃
- 酸
  - 将要行动时，丢弃上一个物品1个
- 绿叶
  - 含标签
  - 被力信号激活时，若激活实体不含有同标签红花，就添加同标签红花。
- 红花
  - 含标签
  - 若实体不在含同标签绿叶的实体上，则丢弃自身，并强度6激活所有含同标签绿叶的实体
- 陷阱
  - 被损伤时，将自身类型改为尖刺

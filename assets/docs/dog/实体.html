实体介绍

实体是游戏的基本元素，是一个大类，下分很多小类。

属性列表 

{
  id:1//序号
  pos:[1,2,3]//位置，x,y,z坐标
  type:"player"//种类
  active:false//激活状态，true已激活,false未激活
  dir:0//方向，0123:上左下右
  inventory:[{*item*}*15]//物品栏
}

序号

序号，每个id对应一个实体，实体被摧毁也不会释放。
创建时，使用整型位置: M^2*z+M*y+x作为序号。

物品栏

每实体都有物品栏，内有15个位置用来放物品。

工作

实体被激活时会工作，工作效果取决于实体种类与物品。

行动

改动过

在回合开始时自动工作就是行动，对应实体叫可行动实体。

移动

移动指的实体的位置发生连续的变化。注意实体的位置改变了很多，而不经历中间一步一步的过程，我们称之为传送。

实体尝试移动到一个空的位置时实体会移动成功，自身的位置变为新的位置。而当移动失败时实体停留在原位置。实体尝试移动到地图边界之外，会移动失败。实体尝试移动到一个已实体的位置时，当另一实体可推动，则会尝试按这个方向移动另一实体；当另一实体不可推动时，会移动失败。
这使得可以一次性推动任意多的一整排连在一起实体，仅取决于地图的大小。

传送

传送指的实体的位置直接改变为另一个位置，而不会和中间路径上实体产生交互。传送仅能传送到空位置，若没有特殊说明，传送到已实体的位置会导致传送失败实体停留在原位。

实体列表

行动类

以下实体能够行动。

  • 玩家
    • 方向操作则向对应方向位移，选择交互键则停留
    • 然后向脚下的实体传递强度为7的力信号
  • 矿车
    • 自动向对应方向位移
    • 然后向脚下的实体传递强度为7的力信号
  • 箱子
    • 停留
    • 然后向脚下的实体传递强度为7的力信号

基础类

效果比较基础，效果无说明默认是工作效果。

  • 尖刺
    • 有磁石：若为磁石方向的逆方向的实体激活，摧毁激活者
    • 否则：摧毁激活者
  • 屏障
    • 被动
      • 有磁石：使尝试逆方向进入此格的实体停留
      • 无磁石：使尝试进入此格的实体停留
  • 压力板
    • 有磁石且无标签：激活面对的实体
    • 有标签：激活同标签的实体
  • 弹射板
    • 将激活者传送到同标签实体的上方，除非该位置被遮挡
  • 注册站
    • 被力信号激活时：将激活者的标签的标识码给面对的实体
    • 被电信号激活时：给面对的实体强度1的信号
  • 传送带
    • 有磁石：
      • 磁石有单一方向：使激活实体向此方向位移
      • 否则：使实体行走
    • 无磁石：
      • 使激活实体向此传送带的方向位移
  • 灰烬
    • 损伤灰烬自身

电路类

不能接收面对实体的电信号，仅向面对实体传递(输出)电信号。

  • 激光传感器
    • 激活者为中心，检查其方向的射线是否有实体。如果有，输出强度1，否则输出强度0
  • 检票口
    • 接受到强度大于6的力信号：检验激活者的物品栏是否含有与此检票口物品栏中相同的宝石且数量相等或更多，是则输出1，否则输出0
  • 取反器
    • 仅被电信号激活
    • 有方向
    • 强度为0：输出正电
    • 强度为1：输出负电
  • 接线口

    • 太麻烦了这个，待商讨是否需要加入

    • 仅被电信号激活
    • 有方向
    • 若此为第2次接受电信号，若有强度1的信号则输出1，否则输出0
  • 电闸

    • 待商讨是否需要加入

    • 强度为1：相同标签的实体保险丝数量+1。若其保险丝数量不少于其保险丝空位的数量，输出1，否则输出0

远程类

通过其方向的射线来远程对实体操作。

  • 风扇
    • 其射线经过的第一个实体向射线方向位移
  • 坑洞
    • 其射线的经过第一个实体向射线反方向位移
  • 炮台
    • 伤害射线经过的第一个实体与特定方块进行联动

联动类

响应其他物体

  • 门
    • 被动
      • 若试图进入的实体不含钥匙，则不可进入
    • 工作
      • 丢弃实体的一个钥匙
  • 森林
    • 若激活实体含电锯，则损伤自身(森林)
  • 沙尘暴
    • 若场上没有同标签的实体(一般搭配森林)，则损伤面对的实体
  • 机场
    • 将激活实体传送到同标签的降落处，不唯一则不生效
  • 防空炮台
    • 强度为6：损伤激活者

其他类

  • 空白
    • 无工作效果，一般也没有物品栏
  • 家园
    • 胜利
  • 钟楼

    • 可能不使用

    • 切换到上一个关卡
  • 补给站
    • 6以上的力信号：激活实体获得补给站的物品栏内容
  • 电磁铁
    • 6以上的力信号：丢弃个数相等的所有的激活实体拥有的本方块物品栏的物品

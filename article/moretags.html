<section> 标签

<section> 是HTML5引入的语义化标签，用于定义文档中的节或段，通常包含一个标题和内容。

使用场景：

  • 将相关内容分组在一起
  • 表示文档中的独立部分
  • 与<article>区别：<article>用于独立内容，<section>用于主题分组
<section> <h2>章节标题</h2> <p>这是章节的内容...</p> <p>更多内容...</p> </section> <section> <h2>另一个章节</h2> <p>这是另一个章节的内容...</p> </section>
<>

<meta> 标签

<meta> 标签用于提供关于HTML文档的元数据，不会显示在页面上，但对机器（如浏览器、搜索引擎）可读。

常见meta标签类型：

属性 值 描述
charset 字符编码（如UTF-8） 定义文档的字符编码
name="viewport" width=device-width, initial-scale=1.0 控制移动设备上的视口行为
name="description" 页面描述文本 提供页面内容的简短描述
name="keywords" 逗号分隔的关键词列表 提供与页面内容相关的关键词
name="author" 作者姓名 定义页面的作者
http-equiv="refresh" 时间;url=地址 定期刷新或重定向页面
<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta name="description" content="免费在线教程"> <meta name="keywords" content="HTML,CSS,JavaScript"> <meta name="author" content="张三"> <meta http-equiv="refresh" content="30"> </head>
<>

<script> 标签

<script> 标签用于定义客户端JavaScript脚本，可以包含脚本语句或通过src属性指向外部脚本文件。

主要属性：

  • src: 指定外部脚本的URL
  • type: 规定脚本的MIME类型（默认text/javascript）
  • async: 异步执行脚本（仅对外部脚本有效）
  • defer: 延迟脚本执行到文档解析完成

使用场景：

1. 内联JavaScript代码：

<script> document.getElementById("demo").innerHTML = "Hello JavaScript!"; </script>

2. 引用外部JavaScript文件：

<script src="myscript.js"></script>

3. 使用async属性异步加载：

<script src="demo_async.js" async></script>

4. 使用defer属性延迟执行：

<script src="demo_defer.js" defer></script>
<>

其他重要标签

<link> 标签

定义文档与外部资源的关系，最常见的是链接CSS样式表。

<link rel="stylesheet" type="text/css" href="mystyle.css">

<style> 标签

用于定义HTML文档的内部CSS样式。

<style> body {background-color: powderblue;} h1 {color: blue;} </style>

<div> 标签

通用容器，用于分组和样式化内容块。

<div class="container"> <h2>这是一个div容器</h2> <p>包含一些内容...</p> </div>

<span> 标签

内联容器，用于标记文本的一部分或文档的一部分。

<p>这是一段文本，其中<span class="highlight">这部分很重要</span>。</p>

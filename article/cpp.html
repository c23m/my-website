C++ 算法常用容器与方法速查（刷题精简版）

本文档专为算法竞赛/刷题整理，只包含最常用的容器、方法及通用算法，略去了实际刷题中极少用到的特性。所有方法均给出完整签名，便于快速查阅。

一、容器概览

类别 容器 特点
顺序容器 vector 动态数组，随机访问，尾部操作快
deque 双端队列，两端操作快，随机访问
list 双向链表，任意位置插入/删除快
string 字符串，可视为 vector<char>
关联容器 set / multiset 有序集合，红黑树，键唯一/可重复
map / multimap 有序映射，键值对，键唯一/可重复
无序关联容器 unordered_set / unordered_map 哈希集合/映射，平均 O(1) 操作
容器适配器 stack 栈，LIFO
queue 队列，FIFO
priority_queue 优先队列，默认最大堆

二、通用容器方法

以下方法几乎所有容器都支持。

方法签名 说明
size_type size() const 元素个数
bool empty() const 是否为空
void clear() 清空所有元素
iterator begin() 首迭代器
iterator end() 尾后迭代器
reverse_iterator rbegin() 反向首迭代器
reverse_iterator rend() 反向尾后迭代器

五、基础语法示例

1. 创建容器

#include <vector>
#include <map>
#include <string>
using namespace std;

// 栈上创建
vector<int> v1;                         // 空vector
vector<int> v2(10, 5);                   // 10个5
vector<int> v3{1, 2, 3, 4};              // 列表初始化

map<string, int> mp;                     // 空map
mp["one"] = 1;                            // 插入

// 堆上创建（使用智能指针，推荐）
auto pv = make_unique<vector<int>>(5, 10);   // C++14

2. 基本操作

// vector 添加/删除
v1.push_back(100);
v1.pop_back();
v1.insert(v1.begin(), 99);
v1.erase(v1.begin());

// 遍历
for (int x : v1) cout << x << " ";
for (auto it = v1.begin(); it != v1.end(); ++it) cout << *it << " ";

// map 查找
if (mp.find("one") != mp.end()) {
    cout << mp["one"];          // 使用[]再次访问
}

3. 算法使用

#include <algorithm>
#include <numeric>

vector<int> a = {5, 2, 8, 1, 9};

// 排序
sort(a.begin(), a.end());               // 升序

// 二分查找（需有序）
bool found = binary_search(a.begin(), a.end(), 8);

// 去重
a.erase(unique(a.begin(), a.end()), a.end());

// 反转
reverse(a.begin(), a.end());

// 累加
int sum = accumulate(a.begin(), a.end(), 0);

// 查找
auto it = find(a.begin(), a.end(), 8);
if (it != a.end()) cout << *it;

4. 迭代器失效注意

  • vector：insert、push_back 可能导致重新分配，使所有迭代器失效；erase 使被删元素及之后迭代器失效。
  • deque：除首尾插入外，其他插入可能导致所有迭代器失效。
  • list / set / map：插入和删除不影响其他迭代器（除指向被删元素的迭代器）。

六、刷题小贴士

  • 常用容器组合：

    • 需要随机访问 → vector
    • 需要两端操作 → deque
    • 需要快速查找 → unordered_map / unordered_set
    • 需要有序键值对 → map
    • 需要自动排序且去重 → set
    • 需要单调栈/队列 → stack / deque
    • 需要求 top k → priority_queue

  • 常用算法组合：

    • 排序 + 二分 → sort + lower_bound
    • 前缀和 → partial_sum 或手动循环
    • 去重排序 → sort + unique + erase
    • 全排列 → next_permutation
    • 最大子数组和 → Kadane 算法（非 STL）
    • 图论 → 手写 BFS/DFS，配合 queue/stack

  • 避免不必要操作：vector 频繁在头部插入应改用 deque；map 的 [] 会插入不存在的键，查找时优先用 find。
